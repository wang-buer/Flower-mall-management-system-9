## 1、项目介绍

鲜花商城管理系统拥有两种角色

管理员：用户管理、分类管理、商品管理、添加鲜花、订单管理、留言管理、新闻管理等

用户：登录注册、购物车、下单、历史订单记录、分类查询商品等


## 2、项目技术

后端框架： Servlet、mvc模式

前端技术：jsp、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：Tomcat 7-10版本均可
- 数据库类型：MySql（5.x和8.x版本都可） 
- maven版本：不需要
- 硬件环境：Windows 或者 Mac OS
